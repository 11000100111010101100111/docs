# Databend

## 支持的版本

Databend v0.9 及以上

## 连接配置

| 参数             | 说明                                                  |
| ---------------- | ----------------------------------------------------- |
| host             | Databend 连接地址                                     |
| user             | 数据库用户名                                          |
| password         | 密码                                                  |
| port             | Databend 服务端口                                     |
| database         | 数据库名称                                            |
| additionalString | 额外的连接串信息，例如 `?xxx=yyy&wait_time_secs=10s&` |
