# 工作台介绍

工作台主要帮助用户快速了解和熟悉产品，并可以通过快速创建入口创建 Agent、连接和任务。

![](../images/workshop.png)

## 快速开始

只需简单 3 步，即可体验 Tapdata 强大、便捷的数据流转能力：

- ① 管理 Agent：单击**创建 Agent**，根据页面引导和本地环境部署 Agent。
- ② 连接管理：单击创建连接，根据页面引导为 Agent 和数据库建立连接。
- ③ 任务管理：一键进入任务创建页面，创建数据同步任务。
